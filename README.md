# Hey, I'm Thomas

I'm a full stack developer, and honestly it started with a video game. I spent way too much of my childhood playing MapleStory, then got it into my head that I wanted to run my own private server — and that rabbit hole is how I learned to code. A web design class in high school and a friend who took the time to teach me did the rest.

These days I build the kind of software where the hard part is hidden: payment processing, real-time gaming, lottery systems — things with a lot going on under the hood that still need to feel simple to whoever's using them. I like living on both sides of that line, modelling the domain on the server with **Laravel** and then building the **Vue 3 / Inertia** frontend that makes it click.

A few things I've worked on:

- **SpadeSocial** — a social gaming platform with a game lobby, a live activity feed, and admin tooling (I built the Mines and Coin Flip games, among other things)
- **BeatLab** — a rhythm game that turns any song into a playable chart, solo or 1v1
- **Mountainside** — a community site for a game server
- A good amount of client work in fintech, gaming, and lottery that I can't show off by name

Which brings me to the elephant in the room: most of my repos here are private client work, so this profile looks quieter than my actual week. If you want the real picture — the projects, the story, the résumé — that all lives on my [portfolio](https://thomaschengdev.laravel.cloud).
